# 📝 TareApp - Gestor de Tareas

**TareApp** es una aplicación de gestión de tareas desarrollada con **React** como parte del curso de Platzi. El enfoque principal fue crear una interfaz limpia, moderna y altamente funcional.

## 🚀 Lo que aprendí en este proyecto:

En este reto puse en práctica conceptos avanzados de React para mejorar la arquitectura de la aplicación:

- **React Portals:** Aprendí a renderizar los modales fuera del nodo principal del DOM (`#root`) usando un nodo específico (`#modal`), manteniendo la jerarquía visual limpia.
- **React Context:** Implementé la API de Contexto para centralizar el estado de la aplicación y evitar el _prop drilling_, permitiendo que componentes como `TodoForm` y `TodoCounter` se comuniquen fácilmente.
- **Componentes Controlados:** Manejo de formularios e inputs sincronizados con el estado de React (`useState`).
- **CSS & Diseño Custom:** Reforcé el uso de clases dinámicas y propiedades como `borderRadius` para lograr bordes curvos y un estilo visual coherente con la identidad de la app.

## 🛠️ Tecnologías utilizadas:

- React.js
- CSS3 (Diseño responsivo y customizado)
- JavaScript (ES6+)
- LocalStorage (Para persistencia de datos)
