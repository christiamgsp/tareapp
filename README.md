# 📝 TareApp - Gestor de Tareas

**TareApp** es una aplicación de gestión de tareas desarrollada con **React**. El enfoque principal fue crear una interfaz limpia, moderna y altamente funcional.

## 🛠️ Tecnologías utilizadas:

- React.js
- CSS3 (Diseño responsivo y customizado)
- JavaScript (ES6+)
- LocalStorage (Para persistencia de datos)
